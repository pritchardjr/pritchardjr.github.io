The Atomics class requires the use of the Gnu Scientific Library (GSL) for calculating the Wigner 6-j function. This, freely available, package is available for download from here. Interfacing with the GSL requires editing the Makefile to point to the directories containing the GSL header and library files.

To compile the code:

  • Install the GSL to a convenient place
  • gunzip cascade.tar.gz
  • tar -xvf cascade.tar
  • Edit the Makefile
  • make

    • Details of the calculation can be found in the paper "Descending from on high: Lyman series cascades and spin-kinetic temperature coupling in the 21 cm line" by J.R.Pritchard and S.R.Furlanetto.
